# Mobile Usage Behavior Analysis (India)

## Project Overview
This project analyzes mobile usage behavior in India using Microsoft Excel. 
The dataset includes user activity, screen time, and app usage patterns.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Cleaning

## Key Insights
- Average screen time of users
- Most used app categories
- Usage patterns by age group
- Daily mobile activity trends

## Files in Repository
- Mobile_Usage_Behavior_Final_India.xlsx → Excel analysis file
- ProjectDashboard.png → Screenshot of Excel dashboard
